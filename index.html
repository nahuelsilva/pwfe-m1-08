Aprendiendo Git

(y no morir en el intento..)

git es un software de control de version diseñado por Linus torvalds.

Pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando éstas tienen un gran número de código fuente. Su propósito es llevar registro de los cambios en archivos de computadora y coordinar el trabajo que varias personas realizan sobre archivos compartidos
